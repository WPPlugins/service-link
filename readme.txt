== Installation ==

Download, Upgrading, Installation:

Upgrade

1. First deactivate Service_link
1. Remove the `Service_link` directory

**Install**

1. Unzip the `Service_link.zip` file. 
1. Upload the the `Service_link` folder (not just the files in it!) to your `wp-contents/plugins` folder. If you're using FTP, use 'binary' mode.

**Activate**

1. In your WordPress administration, go to the Plugins page
1. Activate the Service_link plugin and a subpage for Service_link will appear
   in your Options menu.
